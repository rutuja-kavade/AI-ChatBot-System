🤖 AI ChatBot System

An interactive desktop-based AI ChatBot application developed using Java Swing and Object-Oriented Programming (OOP) concepts.

🚀 Features
Interactive ChatBot for user conversations
Modern and user-friendly GUI
Robot logo and custom background image
Timestamp-based messaging
Chat management functionality
Clean and responsive desktop interface
🛠️ Technologies Used
Java
Java Swing
Object-Oriented Programming (OOP)
